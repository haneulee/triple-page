## ⚙️ Installation

#### clone

```bash
$ git clone https://github.com/haneulee/triple-page.git
$ cd triple-page
```

#### npm

```bash
$ npm install
$ npm run start
```

#### yarn

```bash
$ yarn install
$ yarn start
```
